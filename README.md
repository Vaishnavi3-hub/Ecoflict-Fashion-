# 🌱 Ecoflect Earthwear

Ecoflect Earthwear is a sustainable fashion e-commerce platform designed to promote eco-friendly clothing and conscious consumer choices. The platform helps users explore sustainable products while learning about their environmental impact.

## 🚀 Features

- Modern and responsive user interface
- Sustainable fashion product catalog
- Shopping cart functionality
- Sustainability Calculator
- Customer testimonials section
- FAQ page
- Contact page
- Newsletter subscription
- AI Chatbot support
- Eco-friendly brand awareness

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- HTML5
- CSS3
- Tailwind CSS
- Framer Motion

### Backend & Database
- Supabase

### Tools
- Vite
- Git
- GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ecoflect-earthwear.git
```

Navigate to the project folder:

```bash
cd ecoflect-earthwear
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🌍 Project Objective

The objective of Ecoflect Earthwear is to encourage sustainable fashion practices by providing users with environmentally conscious clothing options and educating them about their ecological footprint.
