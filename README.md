# Ecoflect - Sustainable Fashion Website

## Project Overview
Ecoflect is a responsive WordPress website designed for an eco-friendly fashion startup focused on sustainable clothing solutions and a modern shopping experience.

## Features
- Responsive Design
- Hero Section
- Featured Products
- Sustainability Section
- Testimonials
- Product Showcase
- Contact Form with Validation
- Database Storage of Contact Submissions
- Custom Post Type (Eco Products)
- SEO Optimization

## Technologies Used
- WordPress
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## Pages
1. Home
2. About Us
3. Product Showcase
4. Contact

## Custom WordPress Feature
Custom Post Type: Eco Products

## Installation
1. Clone the repository.
2. Import the WordPress files.
3. Configure the database.
4. Activate the theme/plugin.
5. Run the website locally.
